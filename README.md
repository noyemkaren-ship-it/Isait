## ✨ Особенности проекта

**Isait** — это не просто сайт-визитка. Это полноценная современная веб-платформа с:

- Красивым премиальным дизайном в тёмной теме
- Динамической загрузкой услуг с сервера
- **Мощной и очень красивой формой заявок**
- Полноценным REST API на TypeScript
- Защищённой админ-панелью
- Работой с файлами (загрузка изображений)

---

## 🖥️ Frontend (React + TypeScript)

**Технологии:**
- **React 18** + **TypeScript**
- **Vite** (молниеносная сборка)
- **Tailwind CSS 4** — идеальный современный дизайн
- Полностью типизированный код
- Красивые микро-анимации и плавные переходы

**Что реализовано:**

### 🔥 Главная фишка — Форма заявки (Bid Form)
- Стильный премиум-дизайн (тёмная тема + циановые акценты)
- Поля: Имя, Email, Телефон, Комментарий
- Переключатель "Предпочтительный способ связи" (Email / Телефон)
- Полная клиентская валидация
- Красивые состояния загрузки, успеха и ошибок
- Мгновенная отправка данных на бэкенд
- Автоматический сброс формы после успешной отправки

### Другие возможности:
- Динамическая секция **"Наши услуги"** (загружается с API)
- Современный Navbar с плавной прокруткой
- Мощный Hero-блок
- Полностью адаптивный дизайн
- Профессиональная типографика и spacing

**Frontend выглядит дорого и современно** — именно так, как ожидают клиенты от IT-компании уровня.

## ⚙️ Backend (Node.js + TypeScript)

**Технологии:**
- **Express.js** + **TypeScript**
- **Drizzle ORM** + **SQLite**
- **Multer** — загрузка изображений
- JWT-like авторизация через cookies (`decrypt`)
- Чистая архитектура с роутами

### Что реализовано:

### 📦 Эндпоинты

#### Публичные:
| Метод | Endpoint          | Описание                    |
|-------|-------------------|-----------------------------|
| GET   | `/services`       | Получить все услуги         |
| GET   | `/examples`       | Получить примеры работ      |
| POST  | `/bid`            | **Создать заявку** (главный) |

#### Админские (требуют роль `admin`):
| Метод | Endpoint                    | Описание                     |
|-------|-----------------------------|------------------------------|
| POST  | `/admin/service`            | Добавить услугу + фото       |
| DELETE| `/admin/service/:id`        | Удалить услугу               |
| POST  | `/admin/example`            | Добавить пример работы       |
| DELETE| `/admin/example/:id`        | Удалить пример               |
| GET   | `/bids`                     | Посмотреть все заявки        |
| DELETE| `/admin/bid/:id`            | Удалить заявку               |

### Крутые фичи бэкенда:

- **TypeScript** — строгая типизация, никаких `any` где не нужно
- **Drizzle ORM** — современный, типобезопасный ORM
- **Multer** с валидацией (только изображения, максимум 5MB)
- **Защита админки** через `requireAdmin` middleware
- Чистые роуты и хорошая обработка ошибок
- Логирование важных действий (`print`)
- Полная работа с файлами (`/public/uploads`)
- Встроення админ панель прямо в beckend для которой нужный права доступа ! (`/admin/panel`) а получить прва вот тут ("/login"/)

**Backend получился надёжным, расширяемым и профессиональным.**

## 🛠️ Быстрый старт

### Frontend

```bash
cd client
npm install

# Установи Tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Замени файлы на готовые:
# - src/App.tsx
# - src/index.css

npm run dev
```

### Backend

```bash
cd server          # или где у тебя backend
npm install
npm run dev
```

**Важно:** В `App.tsx` укажи правильный адрес своего сервера:
```ts
const API_URL = 'http://46.253.132.225:5000'
```
